## ‚ö° If Something Goes Wrong

### Website Won't Start?
1. Make sure you ran `npm install` first
2. Check if another program is using port 3000
3. Try running `npm run dev` again

### Photos Not Showing?
1. Check if the photo uploaded successfully (no error message)
2. Refresh the page
3. Make sure the photo file isn't too large (max 10MB)

### Can't Log In?
1. Double-check the email and password
2. Make sure CAPS LOCK is off
3. Try refreshing the page
